# Objectives

- Identify distributions using a Venn diagram
- Convert a graphical model to a factorized distribution
- $1 = \sum p(x)
- Understand that $p(x, y) = p(x \mid y)p(y)$
- Understand $p(x) = \sum_y p(x, y)$
- Understand $p(x \mid y) = p(x, y) / p(y)$
